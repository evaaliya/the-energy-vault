# External integrations used in this project

This project integrates with:

1) World App ecosystem
2) World ID verification
3) XMTP chat
4) AgentKit (future)

All integration documentation is located in /docs folder.

When implementing features, ALWAYS read relevant docs first.

Priority order:
- For identity → read docs/world/*
- For chat → read docs/xmtp/*
- For agent infra → read docs/agentkit/*
